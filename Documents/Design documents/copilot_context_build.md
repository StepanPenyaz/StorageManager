# Copilot repository context preparation

Ensure the repository is prepared for efficient Copilot usage and automated tasks.

Actions:
- Update `Documents/Copilot/copilot_instructions.md` to match the project stack and code style.
- Update `Documents/Design documents/project_structure.md` to reflect the current repository layout.
- Update `Documents/Design documents/domain_requirements.md` to match the current domain model and requirements.
- Prepare `Documents/Copilot/copilot_context.md` with the key repository context for future Copilot runs.

Suggested `copilot_context.md` contents:
- High-level architecture and module responsibilities
- Important files and their locations
- Build, run, and test commands
- Coding conventions and linters/formatters
- Common tasks, scripts, and CI commands
- Known limitations and TODOs
